# Sql Zero To Hero - 30 Days Roadmap

## Overview

This repository contains a structured 30-day SQL roadmap with: 
- daily learning goals 
- SQL practice exercises 
- mini projects every day 
- GitHub-friendly documentation 
- portfolio-ready projects
  
By the end of this roadmap, you will be able to: 

- write complex SQL queries 
- analyze business data 
- clean and transform datasets 
- work with joins and subqueries 
- use window functions 
- optimize queries 
- build real-world SQL projects
---
# Tools
- SSMS (SQL Server Management Studio)
---
# Skills Covered
### Beginner Level
- SELECT statements 
- filtering data 
- sorting 
- aggregate functions 
- GROUP BY 
- HAVING 
- basic joins 
### Intermediate Level 
- subqueries 
- CTEs 
- window functions 
- views 
- indexes 
- stored procedures 
- query optimization 
### Portfolio Projects 
- sales analysis 
- customer analysis 
- employee database analytics 
- ecommerce reporting 
- business KPI dashboards
